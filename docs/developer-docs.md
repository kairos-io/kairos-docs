# Developer docs

The deep reference tree — the material you need once you have picked a persona in [Start here](/docs/start/) and a product in [Products](/docs/products/), and want to actually build, boot, install, upgrade, or extend Kairos.

Everything below is grouped by function:

- **Installation** — put Kairos on real hardware, VMs, cloud, or edge devices.
- **Upgrade** — move a running node forward safely (A/B, Trusted Boot, Operator-driven).
- **Architecture** — how the immutable model, providers, network, and boot chain fit together.
- **Advanced** — sysexts, partitions, encryption, bundles, confidential computing, secureboot cert rotation.
- **Reference** — CLI (AuroraBoot, kairosctl), Kairos Factory, artifact naming, image matrix, configuration schema, FAQ, troubleshooting.
- **Examples** — end-to-end scenarios: airgap, FIPS, MetalLB, KubeVirt, cluster setups.
- **Development** — contributor workflow, debugging station, NVIDIA build notes.
- **Media** — talks and external material.
- **Announcements** — release notes and project announcements.

Pick a section from the sidebar. If you were sent here from a persona or product page, follow the specific link — this page is only an index.
